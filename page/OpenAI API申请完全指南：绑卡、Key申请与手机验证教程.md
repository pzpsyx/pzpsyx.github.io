## 3. 绑定支付方式

1. 进入Settings -> Billing
2. 选择"Add payment details"
3. 点击"Set up paid account"
4. 选择账户类型（个人/企业）
5. 填写支付信息：
   - 卡号
   - 有效期
   - CVV码
   - 账单地址和邮编
6. 选择充值金额（5-50美元）
7. 设置自动充值（建议开启）
