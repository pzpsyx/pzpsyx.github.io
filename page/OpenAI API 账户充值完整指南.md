## 充值步骤详解

1. **登录账户**

   - 使用您的账户凭据登录

2. **进入账单页面**
   - 导航至"Settings"(设置)
   - 点击"Billing"(账单)选项

3. **添加支付方式**

   - 填写支付信息

4. **选择充值金额**
   - 可选范围：$5-$50
